🐟 Browser Private Pool Coordinator

This is the missing own-browser pool layer. It uses BroadcastChannel/local browser messaging so tabs you open can register as workers, receive non-overlapping nonce ranges, report hash-rate, and return found candidates to the Control Nest. This does not recruit unknown visitors; it coordinates your own open browser sessions under this same page/origin.

₿ Bitcoin Network Adapter

Auto-submit only triggers when a hash meets the real Bitcoin network target, not ordinary 0000 practice shares.
