₿ Bitcoin Network Adapter

Connects the Control Nest to a Bitcoin Core JSON-RPC endpoint, pulls real block templates, builds block-reward work paying to the stamped BTC address, sends nonce ranges to the existing browser workers, keeps SSHT harmonic scoring active during header hashing, and submits a solved block back through submitblock. Auto-submit only triggers when a hash meets the real Bitcoin network target, not ordinary 0000 practice shares.
